## Default Permission

Default permissions for the scan plugin

#### This default permission set includes the following:

- `allow-scan-start-menu`
- `allow-scan-uwp-apps`
- `allow-scan-folder`
- `allow-import-scanned-apps`
- `allow-auto-scan-on-start`

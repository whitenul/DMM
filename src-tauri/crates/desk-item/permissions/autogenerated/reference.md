## Default Permission

Default permissions for the item plugin

#### This default permission set includes the following:

- `allow-get-items-by-category`
- `allow-create-item`
- `allow-update-item`
- `allow-launch-item`
- `allow-move-item`
- `allow-reorder-items`
- `allow-delete-item`
- `allow-toggle-pin-item`
- `allow-batch-delete-items`
